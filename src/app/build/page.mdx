# Build

Prepare the client consumable file from the command line.

## Configuration

See [Configuration](/configuration) for options.  From the command line, execute the fastifried/consume/index.js file with the argument 'config' providing a relative path to the configuration file.
```json {{ 'title': 'Build Process Example' }}
      {
        "devDependencies": {
            "fastifried": "1.0.0"
        },
        "scripts": {
            "build:prod": "node ./path/to/fastifried/consume/ config=./path/to/config/fastifry.config.js"
        }
      }
```

The script will export the Fastifried Fetch wrapper and paths.json file to the location specified in your config.
